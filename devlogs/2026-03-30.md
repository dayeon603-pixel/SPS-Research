# Dev Log — 2026-03-30: SPS Estimation Script + Configs

## What Was Built
- `experiments/estimate_sps.py`: full evaluation pipeline
  - Loads any HuggingFace model
  - Runs specified transformation families
  - Computes SPS with confidence intervals
  - Outputs results table + JSON

## Configuration
- `configs/default.yaml`: model list, transformation families, hyperparameters

## Scaling Concern
Full JVP computation on large models (GPT-2 1.5B) is very expensive. Need to investigate:
- Approximation via random projections
- Subset sampling (compute on 500 examples, extrapolate)
- Whether sensitivity estimate from a small subset generalizes
