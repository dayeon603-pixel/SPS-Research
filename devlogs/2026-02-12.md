# Dev Log — 2026-02-12: Output-Only Consistency Metric (Failed)

## Goal
Measure d_Y(f(x), f(Tx)) in output space without constraining what T does.

## What Happened
- Seemed more principled than random noise — at least comparing real outputs
- Problem: a model that memorizes training data appears perfectly stable
- It returns the same prediction for original and any paraphrase of a training point
- But fails on novel paraphrases — which the output-only metric never detects
- Output-space metrics collapse information and hide the exact failure mode I care about

## Decision
Sensitivity needs to be measured at the **representation level**, not just the prediction level. This pushed me toward the Jacobian reformulation: measure how much internal representations change under structured perturbations.

## Key Insight
```
Output consistency ≠ Representation stability
A model can give the same answer for the wrong reasons.
```
