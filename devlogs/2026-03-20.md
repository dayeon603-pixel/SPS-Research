**Implication for my framework:** The SPS depth profile (Definition 7 in `theory/definitions.md`) would be interesting to measure. Running SPS on each layer separately would tell you where semantic stability is highest in the network.

## [09:22 UTC] Implementation Session — First BERT vs RoBERTa Comparison

### State at Session Start

GPU available. Reading done. Setup for the first real cross-architecture comparison.

Setup:
- `bert-base-uncased`: 110M params, 12 layers, $d = 768$ hidden
- `roberta-base`: 125M params, 12 layers, $d = 768$ hidden
- Dataset: SST-2 test split (1,821 examples)
- Transformation family: `SynonymSubstitutionFamily`, substitution_prob=0.15, max_substitutions=3
- Also: back-translation via MarianMT (Helsinki-NLP/opus-mt-en-de + opus-mt-de-en)
- Config: $\varepsilon = 0.1$, m=32 transforms, seed=42

Back-translation pipeline:

```python
def back_translate(
    text: str,
    en_de: MarianMTModel,
    de_en: MarianMTModel,
    tokenizer_en_de: MarianTokenizer,
    tokenizer_de_en: MarianTokenizer,
) -> str:
    tokens = tokenizer_en_de(text, return_tensors="pt", truncation=True, max_length=128)
    with torch.no_grad():
        de = en_de.generate(**tokens)
    de_text = tokenizer_en_de.decode(de[0], skip_special_tokens=True)
    tokens2 = tokenizer_de_en(de_text, return_tensors="pt", truncation=True, max_length=128)
    with torch.no_grad():
        en = de_en.generate(**tokens2)
    return tokenizer_de_en.decode(en[0], skip_special_tokens=True)
```

### Results / Observations

**Main results table (SST-2 test split, n=1,821):**

| Model | Accuracy | SPS (synonym) | SPS (back-trans) | Mean CLS cosine sim |
|---|---|---|---|---|
| bert-base-uncased | 91.2% | 0.717 | 0.584 | 0.881 |
| roberta-base | 94.1% | 0.814 | 0.673 | 0.921 |

**Key observations:**

1. RoBERTa has higher accuracy (94.1% vs 91.2%) and higher SPS on both transformation families. The pattern holds.

2. **SPS gap > accuracy gap.** Accuracy difference: 2.9 pp. SPS (synonym) difference: 0.097. This is the result I wanted to see: SPS captures a dimension of model quality that accuracy doesn't.

3. **Back-translation SPS systematically lower than synonym SPS** for both models (BERT: 0.584 vs 0.717; RoBERTa: 0.673 vs 0.814). Back-translation introduces more surface variation while preserving meaning — the model sees larger perturbations, consistent with family monotonicity (Definition 1: larger family = lower SPS).

4. **CLS cosine similarity** (0.881 vs 0.921) shows RoBERTa representations more stable, but the magnitude of this difference (0.040) is smaller than the SPS difference (0.097). SPS is more discriminative.

**Sensitivity distribution (synonym SPS):**

| Percentile | BERT-base | RoBERTa-base |
|---|---|---|
| 10th | 0.041 | 0.029 |
| 25th | 0.112 | 0.082 |
| 50th (median) | 0.283 | 0.191 |
| 75th | 0.482 | 0.351 |
| 90th | 0.721 | 0.529 |
| 99th | 1.247 | 0.891 |

Both distributions right-skewed. BERT has a heavier tail — more high-sensitivity examples. The 99th percentile gap (1.247 vs 0.891) is larger than the mean gap (0.341 vs 0.244).

## [17:00 UTC] Whiteboard Session — What the SPS vs Accuracy Decoupling Means

### State at Session Start

Results in hand (preliminary). Quick theory session on what the SPS-accuracy decoupling implies.

### Work Completed

**The key empirical pattern:**

$\Delta \text{accuracy} = 2.9\%$ vs $\Delta \text{SPS} = 0.097$.

If SPS and accuracy were measuring the same thing, their relative differences would be similar. The fact that the SPS gap is proportionally larger than the accuracy gap implies SPS captures a dimension of model quality that accuracy does not.

**What could explain this?**

Hypothesis 1 — RoBERTa has more stable intermediate representations. Both models solve SST-2 with high accuracy, but RoBERTa uses a more stable representation path. This is the "right for the right reasons" vs "right for wrong reasons" distinction from Jacovi & Goldberg.

Hypothesis 2 — RoBERTa's training distribution covers more synonymic variation. Dynamic masking over more data implicitly trains robustness to lexical substitution. BERT's static masking on a smaller corpus doesn't.

Hypothesis 3 — The SPS metric is capturing model capacity rather than semantic structure. Larger/better-trained models may be more stable simply because they have more parameters / training. This would mean SPS is just a proxy for general model quality.

Distinguishing H1 from H3: need a model pair with similar capacity but different SPS. BERT-large (340M params, ~93% accuracy) vs RoBERTa-base (125M, ~94% accuracy): RoBERTa-base has fewer parameters but higher SPS. If this holds after the WordNet fix, H1 is more likely than H3.

**The spectral gap as a diagnostic:**

$\bar{\gamma}_{\text{RoBERTa}} > \bar{\gamma}_{\text{BERT}}$ would mean RoBERTa's Jacobian is more geometrically structured — semantic directions are more orthogonal to the worst-case sensitivity direction. This is a specific geometric mechanism that H1 would predict and H3 would not necessarily predict (a larger model could have the same geometry as a smaller one).

Need to compute $\bar{\gamma}$ for both models. Will do this after the WordNet fix.

### Results / Observations

Theoretical analysis: SPS-accuracy decoupling is consistent with RoBERTa having more stable intermediate representations ("right for the right reasons"). Spectral gap comparison would discriminate between capacity and geometric hypotheses.

### Hypotheses

1. $\bar{\gamma}_{\text{RoBERTa}} > \bar{\gamma}_{\text{BERT}}$ by at least 0.05. If this holds, it suggests the spectral geometry is what changes, not just overall smoothness.

2. BERT-large will have higher SPS than BERT-base (more parameters = smoother representations), but still lower than RoBERTa-base (architecture + pretraining difference dominates parameter count effect).
