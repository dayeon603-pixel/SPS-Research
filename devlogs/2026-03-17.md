# Dev Log — 2026-03-17: README Math Rendering Fix

## What Happened
GitHub CommonMark renderer processes italic markers before the math renderer. So `_{\mathcal{T},\varepsilon}` inside blockquotes got treated as italic markup → broken output.

## Fix
Removed blockquotes around math. Put math on standalone lines. Always preview math-heavy markdown on GitHub before finalizing.

## Lesson
Never put LaTeX subscripts inside GitHub blockquote syntax. The Markdown parser wins over the math renderer.
