## [13:00 UTC] Whiteboard Session ŌĆö Finding and Fixing the Theorem 2 Error

### State at Session Start

Reading session flagged a potential gap in my Theorem 2 statement. Now: attempt to write the proof formally and see exactly where it fails.

### Work Completed

**Original Theorem 2 statement (from Feb 27 notes):**

> Under A1, let $T_\alpha(x) = x + \alpha \delta(x)$ for direction $\delta(x) \in A_x$. Then:
> $$\lim_{\varepsilon \to 0} \mathrm{Sens}_{\mathcal{T},\varepsilon}(f_\theta; x) = \|J_{f_\theta}(x)\|_{A_x}.$$

**Where the proof breaks:**

Step 1: By Fr├®chet differentiability,
$$\lim_{\alpha \to 0} \frac{\|f(x + \alpha \delta(x)) - f(x)\|}{\alpha} = \|J_f(x) \delta(x) / \|\delta(x)\|\|.$$
This is fine for a single fixed $\delta(x)$.

Step 2 (where it fails): I was trying to equate this with $\|J_f(x)\|_{A_x} = \sup_{v \in A_x} \|J_f(x) v\|$.

These are equal only if $\delta(x)/\|\delta(x)\| = \arg\sup_{v \in A_x} \|J_f(x) v\|$ ŌĆö i.e., the specific direction $\delta(x)$ happens to be the worst-case direction. That's not assumed. The LHS is one term; the RHS is the sup over all terms. $\leq$ holds trivially; $=$ requires the sup to be achieved at this specific direction.

**Counter-example constructed to make the gap explicit:**

Let $A_x = \{v_1, v_2\}$ with $\|J_f v_1\| = 0.3$ and $\|J_f v_2\| = 0.7$.

If I parameterize with $\delta(x) = v_1$: LHS $= 0.3$, RHS $= 0.7$. Equality fails.

If I parameterize with $\delta(x) = v_2$ (the worst-case direction): LHS $= 0.7 = $ RHS. But this is circular ŌĆö I assumed $\delta(x)$ achieves the sup to make the equality work.

**The fix: two-part theorem.**

*Part (i) ŌĆö Single direction (correct as stated):*

For fixed $v \in A_x$, $\|v\| = 1$:

$$\lim_{\alpha \to 0} \frac{\|f_\theta(x + \alpha v) - f_\theta(x)\|}{\alpha} = \|J_{f_\theta}(x) v\|.$$

*Part (ii) ŌĆö Restricted operator norm (the actually useful result):*

Under A1 and A2 (compactness of $A_x^{(\varepsilon)}$):

$$\lim_{\varepsilon \to 0} \mathrm{Sens}_{\mathcal{T},\varepsilon}(f_\theta; x) = \sup_{v \in A_x, \|v\|=1} \|J_{f_\theta}(x) v\| = \|J_{f_\theta}(x)\|_{A_x}.$$

**Why Part (ii) needs A2 (compactness):**

The key step is interchanging the $\sup$ and the $\lim_{\varepsilon \to 0}$. In general, $\sup_v \lim f_v \neq \lim \sup_v f_v$. The interchange is valid when convergence is uniform in $v$ ŌĆö which follows from compactness of $A_x$ and the fact that the Fr├®chet remainder $r(\alpha v) / \alpha \to 0$ uniformly in $v$ on compact sets (by A1, which implies uniform differentiability on compact neighborhoods).

So Part (ii) requires both A1 and A2. Part (i) requires only A1.

**Updated `theory/proofs.md`** with the corrected two-part theorem. Added remark on the original error. Committed as `fix(theory): theorem-2 equality vs inequality`.

**Theorem 3 (Lipschitz Stability Bound) drafted in `theory/proofs.md`:**

*Statement:* If $f_\theta$ is globally $L$-Lipschitz, then $\mathrm{SPS}_\varepsilon(f_\theta) \geq e^{-L}$.

*Proof:* By $L$-Lipschitz, $d_\mathcal{Y}(f(x), f(Tx)) \leq L \cdot c(T, x)$. Dividing, $d_\mathcal{Y}/c \leq L$ uniformly. Taking sup and expectations: $\mathbb{E}[\mathrm{Sens}] \leq L$. Applying $\exp(-\cdot)$: $\mathrm{SPS} \geq e^{-L}$.

This bound is weak in practice ŌĆö the Lipschitz constant of BERT over all inputs is large. But Corollary 1 tightens it using $\bar{\gamma}$: $\mathrm{SPS} \geq \exp(-(1-\gamma_0) L)$, which is a factor $e^{-\gamma_0 L}$ tighter.
