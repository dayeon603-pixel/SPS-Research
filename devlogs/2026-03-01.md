# Dev Log — 2026-03-01: Theorem 2 Fix — Equality vs Inequality

## What Happened
Original Theorem 2 claimed:
```
δ(x) [single direction] = ‖J_f(x)‖_{A_x} [sup over all directions in A_x]
```

This is wrong. A single JVP gives the directional derivative for one specific T. The restricted operator norm is the supremum over all directions in A_x. Equality would require the sup to be achieved at exactly that direction — not guaranteed unless A_x is a singleton.

## Fix
Changed `=` to `≤`. The statement becomes:
```
δ(x) ≤ ‖J_f(x)‖_{A_x}
```

This is correct and still useful — it gives an upper bound on single-perturbation sensitivity.

## Lesson
Always check whether you're asserting equality between a single instantiation and an extremal quantity. This is a common proof error in functional analysis.
