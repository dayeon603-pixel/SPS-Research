## [13:30 UTC] Whiteboard Session — Theorem 2 Proof Sketch + Key Derivation

### State at Session Start

Reading session complete. No code. This is a pure theory session: derive the Jacobian connection from Definition 1.

**Goal:** Show that $\lim_{\varepsilon \to 0} \mathrm{Sens}_{\mathcal{T},\varepsilon}(f_\theta; x) = \|J_{f_\theta}(x)\|_{A_x}$.

### Work Completed

**Starting point:**

$$\mathrm{Sens}_{\mathcal{T},\varepsilon}(f_\theta; x) = \sup_{\substack{T \in \mathcal{T} \\ 0 < c(T,x) \leq \varepsilon}} \frac{d_\mathcal{Y}(f_\theta(x), f_\theta(Tx))}{c(T,x)}.$$

**First attempt — wrong:**

Parameterized $T_\alpha(x) = x + \alpha \delta(x)$ for some fixed direction $\delta(x) \in A_x$ and wrote:

$$\lim_{\varepsilon \to 0} \mathrm{Sens}_{\mathcal{T},\varepsilon}(f_\theta; x) \overset{?}{=} \lim_{\alpha \to 0} \frac{\|f_\theta(x + \alpha\delta(x)) - f_\theta(x)\|}{\alpha} = \|J_{f_\theta}(x) \delta(x)\|.$$

The second equality is fine (Fréchet derivative definition). The first equality is wrong. The LHS is a supremum over all $T \in \mathcal{T}$, which is a supremum over all $\delta \in A_x$. The RHS is the JVP norm for one fixed $\delta$. They're equal only if $A_x$ is a singleton.

I wrote this down and almost moved on — didn't catch the error immediately. This gap survives in my notes until March 1. The error: conflating a single directional derivative with the restricted operator norm.

**Correct derivation:**

Split into two parts.

*Part (i) — single direction:* For fixed $v \in A_x$ with $\|v\| = 1$, Fréchet differentiability gives:

$$f_\theta(x + \alpha v) = f_\theta(x) + \alpha J_{f_\theta}(x) v + r(\alpha v), \qquad \frac{\|r(\alpha v)\|}{\alpha} \to 0.$$

Therefore:

$$\lim_{\alpha \to 0} \frac{\|f_\theta(x + \alpha v) - f_\theta(x)\|}{\alpha} = \|J_{f_\theta}(x) v\|. \quad \checkmark$$

*Part (ii) — restricted operator norm:* The sup over $\mathcal{T}$ equals the sup over $A_x$ directions in the $\varepsilon \to 0$ limit. To interchange the sup and the limit, need uniform convergence of the Fréchet remainder on $A_x$, which requires $A_x$ compact.

**Assumption A2 (needed here):** $A_x^{(\varepsilon)} := \{(Tx - x)/\|Tx - x\| : T \in \mathcal{T}, 0 < c(T,x) \leq \varepsilon\}$ is compact in $\mathbb{S}^{d-1}$. If $\mathcal{T}$ contains smooth families, the set of normalized tangent directions $A_x = \lim_{\varepsilon \to 0} A_x^{(\varepsilon)}$ is a closed subset of the sphere, hence compact.

With A2:

$$\lim_{\varepsilon \to 0} \sup_{v \in A_x^{(\varepsilon)}} \frac{\|f_\theta(x + \varepsilon v) - f_\theta(x)\|}{\varepsilon} = \sup_{v \in A_x} \|J_{f_\theta}(x) v\| = \|J_{f_\theta}(x)\|_{A_x}.$$

This is the correct statement of Part (ii). And this is not the same as Part (i) — the two parts are separate claims that need to be stated separately.

**Today's error in my draft:** I wrote a single unified theorem that conflated (i) and (ii). The theorem statement says "$\lim_{\varepsilon \to 0} \mathrm{Sens} = \|J\|_{A_x}$" but the proof only shows this for a fixed direction, which is a much weaker claim. This goes unnoticed until March 1.

**Why the spectral gap follows:**

Once we have $\lim_{\varepsilon \to 0} \mathrm{Sens}_{\mathcal{T},\varepsilon}(f_\theta; x) = \|J_{f_\theta}(x)\|_{A_x}$, the natural comparison is with the full spectral norm $\sigma_{\max}(J_{f_\theta}(x))$. The gap:

$$\gamma(f_\theta, \mathcal{T}; x) := \sigma_{\max}(J_{f_\theta}(x)) - \|J_{f_\theta}(x)\|_{A_x} \geq 0.$$

When $\gamma = 0$: the maximally sensitive direction of the Jacobian is a semantic direction. The model is maximally sensitive precisely along the directions it should be invariant to. Bad.

When $\gamma$ is large: the semantic directions are geometrically orthogonal to (or far from) the maximally sensitive direction. The model's worst-case sensitivity is not along semantic axes. Good.

The normalized version $\bar{\gamma} = 1 - \|J\|_{A_x} / \sigma_{\max}(J) \in [0,1]$ has a clean interpretation and was easy to defend.

### Results / Observations

Theorem 2 structure identified (two parts, with gap in my current draft that I haven't caught yet). Assumption A2 (compactness of $A_x$) identified as load-bearing. Spectral gap definition sketched.

Key formula for future reference:

$$\bar{\gamma}(f_\theta, \mathcal{T}; x) = 1 - \frac{\|J_{f_\theta}(x)\|_{A_x}}{\sigma_{\max}(J_{f_\theta}(x))}, \quad \bar{\gamma} \in [0,1].$$

### Hypotheses

1. The compactness assumption A2 is the load-bearing assumption for Part (ii) of Theorem 2. Without it, the interchange of sup and limit fails.
2. The spectral gap provides a strictly tighter lower bound on SPS than the plain Lipschitz bound — this is Corollary 1.
3. The synonym directions should cluster in a low-dimensional subspace of $\mathbb{R}^{768}$ — semantic axes don't span the full space.

## [20:00 UTC] Implementation Session — Synonym Direction Extraction Prototype

### State at Session Start

Theory session done. Theorem 2 drafted (with unnoticed gap). Now: make the theory concrete. First implementation step: extract semantic direction vectors from the embedding space.

### Work Completed

**Synonym direction extraction prototype:**

For a token $w$ with synonyms $\{s_1, \ldots, s_K\}$ from WordNet:

```python
def get_synonym_directions(
    token: str,
    tokenizer: PreTrainedTokenizer,
    embedding_layer: nn.Embedding,
    max_synonyms: int = 8,
) -> torch.Tensor:
    """Returns normalized synonym directions. Shape: (K, hidden_dim)."""
    synonyms = get_wordnet_synonyms(token, max_synonyms)
    if not synonyms:
        return torch.zeros(0, embedding_layer.embedding_dim)
    token_id = tokenizer.convert_tokens_to_ids(token)
    syn_ids = [tokenizer.convert_tokens_to_ids(s) for s in synonyms]
    token_emb = embedding_layer.weight[token_id]         # (hidden,)
    syn_embs = embedding_layer.weight[syn_ids]           # (K, hidden)
    directions = syn_embs - token_emb                    # (K, hidden)
    norms = directions.norm(dim=1, keepdim=True).clamp(min=1e-8)
    return directions / norms                            # (K, hidden)
```

This is fine for the framework: $A_x$ is defined as the actual directions induced by $\mathcal{T}$ in embedding space, not an idealized semantic axis. Whatever directions synonym substitution induces, those are the right $A_x$ for the synonym family.
