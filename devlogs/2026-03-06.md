No code. Proof session.

**Original Proposition 1 statement:**

> For any neural network $f_\theta$, $0 < \mathrm{SPS}_\varepsilon(f_\theta) \leq 1$.

**Upper bound ($\leq 1$) — easy:**

$d_\mathcal{Y} \geq 0$ and $c(T, x) > 0$ everywhere, so every term in the sup in Definition 1 is non-negative. Therefore $\mathrm{Sens}_{\mathcal{T},\varepsilon}(f_\theta; x) \geq 0$ for all $x$. Taking expectations preserves non-negativity: $\mathbb{E}[\mathrm{Sens}] \geq 0$. Applying $\exp(-\cdot)$: $\mathrm{SPS} = \exp(-\mathbb{E}[\mathrm{Sens}]) \leq \exp(0) = 1$. Done.

**Lower bound ($> 0$) — first attempt:**

"$\exp(-\mathbb{E}[\mathrm{Sens}]) > 0$ because $\exp(t) > 0$ for all $t \in \mathbb{R}$."

This is wrong. $\exp(-t) > 0$ for all finite $t$. But if $t = +\infty$, then $\exp(-\infty) = 0$. The claim fails if $\mathbb{E}[\mathrm{Sens}] = +\infty$.

**When can $\mathbb{E}[\mathrm{Sens}]$ diverge?**

$\mathrm{Sens}_{\mathcal{T},\varepsilon}(f_\theta; x) = \sup_{T: c(T,x) \leq \varepsilon} d_\mathcal{Y}(f(x), f(Tx)) / c(T, x)$.

If $c(T,x) \to 0$ for some $T$ and $d_\mathcal{Y}(f(x), f(Tx)) \not\to 0$ as $c \to 0$, the ratio diverges. This could happen if $f_\theta$ is discontinuous at $x$ — but Fréchet differentiability (A1) rules this out. Under A1, $d_\mathcal{Y}(f(x), f(Tx)) = O(c(T,x))$ for small $c$, so the ratio stays bounded pointwise.

But pointwise boundedness doesn't give $\mathbb{E}[\mathrm{Sens}] < \infty$ without a uniform bound over $x \in \mathrm{supp}(\mathcal{D})$. If the local Lipschitz constant of $f_\theta$ is unbounded as $x$ ranges over $\mathrm{supp}(\mathcal{D})$, the expectation can diverge even if each individual $\mathrm{Sens}(f; x)$ is finite.

**Resolution: Assumption A4.**

Add explicitly: $\mathbb{E}_{x \sim \mathcal{D}}[\mathrm{Sens}_{\mathcal{T},\varepsilon}(f_\theta; x)] < \infty$.

With A4, $\mathbb{E}[\mathrm{Sens}]$ is a finite non-negative real number, and $\exp(-\mathbb{E}[\mathrm{Sens}]) > 0$ follows.

Without A4, the correct claim is only $\mathrm{SPS}_\varepsilon \geq 0$.

**Three approaches I considered for eliminating A4:**

*Option 1:* Assume $f_\theta$ is globally $L$-Lipschitz. Then $\mathrm{Sens} \leq L$ everywhere, so A4 holds with $\mathbb{E}[\mathrm{Sens}] \leq L < \infty$. Problem: Theorem 3 already uses this hypothesis. Having Proposition 1 also require it would make it a corollary of Theorem 3, not an independent result.

*Option 2:* Assume bounded divergence ($d_\mathcal{Y} \leq M$). Then $\mathrm{Sens} \leq M/c_{\min}$ where $c_{\min}$ is the minimum transformation cost. If $c_{\min} > 0$, A4 follows. For cosine divergence (bounded in $[0, 2]$) and $\alpha \sim \mathrm{Uniform}(0, \varepsilon]$, this works. But it constrains the divergence function choice.

*Option 3:* State A4 as an explicit assumption. Cleanest — Proposition 1 is conditional on A4. Honest about what the proof requires.

**Decision: Option 3.** Updated `theory/definitions.md` with A4 explicitly. Updated Proposition 1 to invoke A3 and A4.

Added remark in proofs.md: "Without A4, only $\mathrm{SPS}_\varepsilon \geq 0$ holds. The strict positivity claim is non-trivial and requires bounded average sensitivity."

Committed as `fix(theory): proposition-1 add integrability assumption A4`.

## [14:00 UTC] Whiteboard Session — Integrability and What A4 Actually Rules Out

### State at Session Start

A4 added to the axioms. But I want to understand it better: what class of models does A4 rule out, and is this a meaningful restriction?

### Work Completed

**When does A4 fail?**

A4 requires $\mathbb{E}_{x \sim \mathcal{D}}[\mathrm{Sens}_{\mathcal{T},\varepsilon}(f_\theta; x)] < \infty$.

Case 1 — model has locally unbounded Jacobian:

Suppose $\mathcal{D}$ puts positive mass on a region where $\sigma_{\max}(J_{f_\theta}(x)) \to \infty$. By Theorem 2(ii), $\mathrm{Sens}(f;x) \to \infty$ there. The expectation diverges.

This can happen for neural nets with ill-conditioned weight matrices (e.g., after training without weight normalization or spectral normalization).

Case 2 — cosine divergence with unit-norm representations:

If $f_\theta$ maps everything to unit vectors ($\|f_\theta(x)\| = 1$ for all $x$), then $d_\mathcal{Y}(a, b) = 1 - a \cdot b \leq 2$. The sensitivity is bounded by $2 / c_{\min}$. A4 holds trivially.

Case 3 — L2 divergence with unbounded representations:

If the representation norm $\|f_\theta(x)\|$ grows without bound as $x$ ranges over $\mathrm{supp}(\mathcal{D})$ (e.g., the model outputs features without any normalization), then $d_\mathcal{Y}(f(x), f(Tx))$ can grow without bound, and the ratio $d_\mathcal{Y}/c$ can be arbitrarily large.

**Practical verdict:** For all experiments I'm running (cosine divergence, BERT/RoBERTa with bounded output norms), A4 holds trivially. It's a technical assumption needed for the proof, not a meaningful restriction in practice. Worth one sentence in the paper: "A4 is satisfied by any model with bounded average Jacobian spectral norm under $\mathcal{D}$, which holds for all standard pre-trained transformers."

**Connection to spectral normalization:**

If $f_\theta$ is spectrally normalized (Miyato et al. 2018 — normalize each weight matrix to have spectral norm 1), then $\sigma_{\max}(J_{f_\theta}(x)) \leq L_{\text{prod}} = \prod_l 1 = 1$. A4 is automatic. Interestingly, spectral normalization is typically used for GAN training stability — the motivation there is different (Lipschitz constraint for discriminator), but the effect is the same: A4 is guaranteed.

**Added to theory notes:** A4 is implied by global Lipschitz (Theorem 3's hypothesis) but independent of A1, A2, A3. The cleanest presentation keeps it as a standalone assumption.
