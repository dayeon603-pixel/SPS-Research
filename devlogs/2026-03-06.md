### Work Completed

**Attempted proof of Proposition 1.**

Original statement:

> **Proposition 1 (Boundedness).** For any neural network $f_\theta$, $0 < \mathrm{SPS}_\varepsilon(f_\theta) \leq 1$.

Upper bound proof:

$\mathrm{Sens}_{\mathcal{T},\varepsilon}(f; x) \geq 0$ since $d_\mathcal{Y} \geq 0$ and $c(T, x) > 0$. So $\mathbb{E}[\mathrm{Sens}] \geq 0$. Applying $\exp(-\cdot)$: $\mathrm{SPS} = \exp(-\mathbb{E}[\mathrm{Sens}]) \leq \exp(0) = 1$. Done.

Lower bound attempt:

"$\exp(-\mathbb{E}[\mathrm{Sens}]) > 0$ because $\exp$ is always positive."

This is wrong. $\exp(-t) > 0$ for any finite $t \in \mathbb{R}$. But what if $t = +\infty$? Then $\exp(-\infty) = 0$.

The claim fails if $\mathbb{E}[\mathrm{Sens}] = +\infty$. And there is no reason in principle why $\mathbb{E}[\mathrm{Sens}]$ should be finite. A model with unbounded Jacobian on a set of positive measure would give $\mathbb{E}[\mathrm{Sens}] = +\infty$ and $\mathrm{SPS} = 0$.

**The fix: add Assumption A4 (Integrability).**

$$\mathbb{E}_{x \sim \mathcal{D}}\bigl[\mathrm{Sens}_{\mathcal{T},\varepsilon}(f_\theta; x)\bigr] < \infty$$

With A4, the expectation is a finite real number, $\exp$ of a finite negative real is strictly positive, and the claim holds.

Updated `theory/definitions.md` with Assumption A4 formally stated. Updated Proposition 1 statement to invoke A3 (measurability) and A4 (integrability) explicitly.

**Checked A4 practically:**

For BERT-base and RoBERTa-base on SST-2, the sensitivity distribution has finite support (CLS cosine divergence is bounded in [0, 2]). So A4 is automatically satisfied for any bounded divergence function. The concern is theoretical — unbounded sensitivity could arise for:
- Models with near-singular Jacobians on a dense set (pathological but possible)
- Adversarial directions that cause catastrophic output shifts
- Very large $\varepsilon$ where the perturbation is no longer local

For all practical experiments in this project (cosine divergence, bounded $\varepsilon$), A4 holds trivially. But it needs to be stated for the theory to be complete.

### Lab Notes

The integrability gap in the original proof is subtle. I knew `exp(-t) > 0` for finite $t$ and assumed that was enough. The missing step is that the expectation of Sens must itself be finite. It's the same gap as claiming "the integral of a positive function is positive" — which is false if the integral diverges.

For the paper, this matters. Proposition 1 is the most basic claim about the framework. A reviewer who knows functional analysis would catch this immediately. Catching it myself now is better than catching it in review.

A4 is actually a reasonable modeling assumption. Any model we'd use in practice has bounded Jacobian spectral norm (it has to, for training to be stable). A4 is satisfied by all practical transformer models under bounded divergence functions. But "satisfied in practice" is different from "follows from the definition" and the proof needs to be explicit.

The question of whether A4 is "natural" (expected to hold for typical models) or a meaningful constraint (rules out pathological models) is worth one sentence in the paper.

### Hypotheses

1. **A4 is automatically satisfied for all bounded divergence functions $d_\mathcal{Y}$.** If $d_\mathcal{Y}(a, b) \leq M$ for some constant $M$, then $\mathrm{Sens}_{\mathcal{T},\varepsilon}(f; x) \leq M / \min_{T \in \mathcal{T}} c(T, x)$. If the minimum transformation magnitude is bounded away from zero (which it is under $\alpha \sim \text{Uniform}(0, \varepsilon]$), then A4 follows.

2. **For L2 divergence on unbounded output spaces, A4 is a non-trivial constraint.** If $d_\mathcal{Y}$ is the L2 norm on $\mathbb{R}^k$ and outputs are unbounded, a model with large-norm representations could violate A4. Worth noting as a limitation when comparing to cosine divergence.
