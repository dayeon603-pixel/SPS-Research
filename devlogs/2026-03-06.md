# Dev Log — 2026-03-06: Proposition 1 — Missing Integrability Assumption

## What Happened
Claimed SPS_ε > 0 (strict inequality). Proof: exp(-E[Sens]) > 0 because exp is always positive.

But exp(-∞) = 0. If the sensitivity expectation diverges to +∞, the claim fails.

## Fix
Added Assumption A4 (integrability): E[Sens] < ∞. Without this, can only claim SPS_ε ≥ 0.

With A4, the strict inequality holds because the expectation is finite → exp(-finite) > 0.

## Impact
A4 is a non-trivial assumption. It means the model must have bounded sensitivity on average — which excludes pathological models with unbounded Jacobians. This is actually a reasonable requirement for any practical model, but it needs to be stated explicitly.
